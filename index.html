Linux Mint


Open Source


O Linux Mint é livre para baixar, usar e compartilhar, sendo livre de custos e de código aberto.


Seguro e Confiável


É seguro e confiável. Graças a uma abordagem conservadora para atualizações de software, um gerenciador de atualização exclusivo e a robustez de sua arquitetura Linux, o Linux Mint requer pouca manutenção.


Dirigido pela Comunidade


Os usuários são encorajados a enviar feedbacks ao projeto para que suas ideias possam ser usadas para melhorar o Linux Mint!

Confortável

O objetivo do Linux Mint é ser um sistema operacional moderno, elegante e confortável, que seja poderoso e fácil de usar, sendo uma das distribuições Linux mais populares e usada por milhões de pessoas.

Fácil de Usar!

Ele trabalha fora da caixa, com suporte multimídia completo e é extremamente fácil de usar.

Baseado no Debian e no Ubuntu, ele fornece cerca de 30.000 pacotes e um dos melhores gerenciadores de software.

Dual Boot

O Linux Mint cumpre as mesmas funções que os sistemas operacionais Microsoft Windows, Mac OS, e o Free BSD, além disso ele pode ser instalado em conjunto com outros sistemas operacionais, podendo portanto, ser configurado automaticamente em um ambiente dual boot ou multi-boot, onde o usuário pode escolher qual sistema operacional será carregado por padrão na inicialização do PC, durante sua instalação. O Linux Mint é um ótimo sistema operacional para usuários comuns e para empresas.

Terminal

Nós concordamos que o Terminal não é o aplicativo com o visual mais atraente do Linux Mint, mas vale a pena saber que ele é, de longe, o mais poderoso. Uma vez que tenha aprendido de fato como usá-lo, verá que não é tão complicado como parecia a primeira vista.

Fácil de Personalizar

Você pode personalizar e modificar o Menu de várias de maneiras. Para isso, dê um clique com o botão direito do mouse no botão “Menu” e selecione “Configurar”.

Escolha entre os milhares de temas e papéis de parede para o sistema ficar do jeito que você quiser.

Loja de Aplicativos

A maneira mais fácil de instalar softwares no Linux Mint é utilizando o Gerenciador de Aplicativos. Voce acha os programas que precisa sem complicação, e o melhor todos são grátis.
